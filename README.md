# pure-url

URL Shortener service written with `MERN` stack: MongoDB, Express, React, Node.js

## Demo

Demo lives here: [https://pure-url.salmi3d.now.sh](https://pure-url.salmi3d.now.sh)

## Setup

```
yarn setup
```

## Run for development

```
yarn dev
```

## Run for production

```
yarn start
```
